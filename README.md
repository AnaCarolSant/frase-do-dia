# Frase do Dia

Pequeno app React Native para mostrar uma "frase do dia" e permitir cadastrar novas frases no Firestore.

## Descrição

Este repositório contém um app que carrega frases do Firestore e permite adicionar novas frases. O objetivo deste README é deixar um espaço para você inserir prints (screenshots) do app em execução e do console do Firebase mostrando o cadastro da frase.

---

## Como executar (rápido)

Pré-requisitos:
- Node.js
- npm ou yarn
- Expo CLI (se estiver usando Expo)

Comandos típicos:

```powershell
npm install
npm start
# ou para abrir no emulador Android
npm run android
```

Substitua pelos comandos do seu fluxo (Expo/React Native CLI) conforme necessário.
